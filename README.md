# Smart Alarm ⏰✨

Stay on time.  
Stay focused.  
Never miss the moment. 🚀

Smart Alarm is a clean desktop alarm app for busy people.  
It lives in your tray.  
It shows up when it matters. 🔔

![Smart Alarm main screen](screenshots/main.png)

## Why you'll love it 💛

- Fast reminders ⚡
- Custom colors 🎨
- Gentle or urgent sounds 🔊
- Snooze when you need it 😴
- Weekly, one-time, and interval alarms 📅
